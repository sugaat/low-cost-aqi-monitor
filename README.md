# low-cost-aqi-monitor
esp32 based aqi monitor for pm, co, temperature and humidity

## overview
a low-cost air quality monitoring system based on esp32. the system measures pm1, pm2.5, PM10 , CO concentration , temperature and humidity.

## Hardware Used
- ESP32 Dev Module
- Planttower PMS7003
- Winsen ZE-15 CO
- DHT 22
- LCD screen
